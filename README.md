# pondering.life

A tiny umbrella site for small, useful tools.

Built with Next.js 16 / React 19. Inline styles, no Tailwind, no extra deps.

## Pages

- `/` — homepage with cards linking to each tool
- `/listing-writer` — landing page for the Poshmark listing writer ([repo](https://github.com/elisemarq/poshmark-lister-web))

## Develop

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Deploy

Pushes to `main` auto-deploy via Vercel. The `pondering.life` domain points
here.
